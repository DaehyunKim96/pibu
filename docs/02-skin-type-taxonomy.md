# 02 · 피부 타입 분류 설계

> 코드: `src/data/skinTypes.ts`

## 설계 의도

기존 "건성/중성/지성" 3분류는 **민감·트러블 맥락을 못 담는다**. 그래서:

- 학술적으로 검증된 **Baumann Skin Type** 체계(4축 → 16타입)를 참고하되,
- pibu의 미션(민감·트러블)에 맞게 **실용적 7개 아키타입**으로 압축했다.

### Baumann 4축 → pibu 4축 매핑

| Baumann | pibu 진단 축 | 의미 |
| --- | --- | --- |
| Oily ↔ Dry | `oil` | 유분 많음(+) / 적음(−) |
| Sensitive ↔ Resistant | `sensitivity` | 자극 반응성 |
| (트러블 별도) | `acne` | 트러블 경향 |
| (홍조 별도) | `redness` | 혈관 반응성 |
| — | `barrier` | 장벽 손상 정도 (긴급도 ↑) |

> Baumann의 "Pigmented/Wrinkled" 축은 v0.1에서 제외(민감 케어에 우선순위 낮음). 추후 추가 가능.

## 7가지 아키타입

| 타입 | 트리거 | 핵심 처방 |
| --- | --- | --- |
| 🌵 건성 민감 | oil↓ + 민감 | 세라마이드·판테놀·히알루론산 / 알코올·SLS·AHA 회피 |
| 💧 지성 민감 | oil↑ + 민감 | 나이아신아마이드·징크·녹차 / 알코올·정유 회피 |
| 🌗 복합 민감 | oil≈0 + 민감 | 존별 케어 + 진정 베이스 |
| 🔴 트러블·여드름 | acne↑ | BHA·아젤라산·나이아신아마이드 / 정유·향료 회피 |
| 🌹 홍조·붉음 | redness↑ | 병풀·아젤라산·녹차 / 알코올·AHA 회피 |
| 🩹 장벽 손상 | barrier↑ (최우선) | 장벽 지질 only, 모든 활성 중단 |
| 🌿 건강·정상 | 민감 낮음 | 항산화·자외선 차단 예방 케어 |

## 진단 로직 (우선순위 분기)

긴급/특이도가 높은 상태를 먼저 가른다 (`classifySkinType`):

```
1. barrier ≥ 3        → 장벽 손상   (가장 시급, 활성 중단 필요)
2. acne ≥ 2           → 트러블·여드름
3. redness ≥ 2        → 홍조·붉음
4. sensitivity < 1    → 건강·정상
5. 그 외(민감) + oil   → 건성/지성/복합 민감
```

### 왜 이 순서인가
- 장벽이 무너진 상태에서 "지성이니 BHA 쓰세요"는 **해롭다** → 장벽을 최상위로.
- 트러블·홍조는 유분 분류보다 **케어 방향을 더 강하게 규정**하므로 먼저 분기.
